# Termux Plus
Extra packages for Termux
# Pacakges available
* liborc (armv7l and aarch64 only)
* libspice (armv7l and aarch64 only)
* Headless QEMU with SPICE (armv7l and aarch64 only)
* ALSA Libs
* libwayland
* Xwayland
* liblorie
* Termux:X11
* Modded MESA
* Apache ANT
